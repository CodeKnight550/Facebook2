## 🧩 广告类型新解
### ① 图片广告  
- 适用场景：新品初曝光 • 品牌理念传递  
- 创意公式：高清大图 + 文案≤25字 +「限时优惠」标识  
- 黄金尺寸：1200×628像素（PC端） / 1080×1080像素（移动端）  

### ② 轮播广告  
- 玩法突破：5张产品GIF动图 × 智能排序  
- 进阶技巧：每张卡片添加商品直达链接  
- 数据观察：第三张卡流失率>52%需调整主推顺序  

### ③ 问卷互动广告  
- 数据金矿：通过8-10题问答精准构建用户画像  
- 奖励机制：问卷填完即得$5优惠券  
- 转化追踪：表单提交+优惠券核销双重转化目标
